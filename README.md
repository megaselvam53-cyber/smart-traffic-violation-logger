🚦 Smart Traffic Violation Logger
📌 Project Overview
Smart Traffic Violation Logger is a simple and user-friendly web application.
It helps traffic authorities record and manage traffic violations digitally.
Each violation is stored in a database and a QR code is generated to quickly view the details.
💻 Technologies Used
Python
Flask
HTML
CSS
Bootstrap
SQLite
⚙️ Features
👮 Admin (Police) Module
Admin Login
Add traffic violations
Generate QR code
View violation history
Filter by vehicle number, date, and status
Mark fine as Paid / Pending
📄 Public Access
Scan QR code
View violation details
Check fine status
📂 Project Structure
run.py → Main file
app/ → Flask app
models.py → Database
routes.py → Backend logic
templates/ → HTML pages
static/ → CSS & QR images
app.db → Database file
⚠️ Current Limitations
Manual data entry required
No online payment system
Works only on local server
No AI-based detection
